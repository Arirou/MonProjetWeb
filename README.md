# TP Technologie du Web - JavaScript

## Description
Ce projet est un TP de révision sur HTML, CSS et JavaScript. Il inclut :
- Un convertisseur de devises (version bouton et version réactive)
- Un formulaire interactif avec vérification des entrées
- Gestion des radios et du volume
- Menu interactif pour afficher/cacher les sections
- Barres de progression simulant un téléchargement
- Stockage temporaire des choix (localStorage)
- 

## Utilisation

Cocher les cases dans le menu pour afficher les sections

Modifier les valeurs dans le convertisseur pour voir la conversion

Interagir avec les radios et le slider pour observer les changements dynamiques

## Pré-requis

Navigateur moderne (Chrome, Firefox, Edge)
JavaScript activé
